Beginner Guides
###############

The **Beginner Guides** section is aimed at first-time users of EduSweep, or those that
have used much older versions in the past. The guides prepare you for running your first
scan task.

.. note::
    These guides assume that EduSweep has already been installed, following the
    instructions provided in the :ref:`installation` section.
